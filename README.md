# Astronomy Data Analysis with Machine Learning

This project explores the application of machine learning techniques to astronomical datasets, with a focus on light curve analysis and classification of celestial objects.

## Overview
- Process and analyze astronomical survey data
- Perform light curve analysis for variable stars and exoplanets
- Apply basic machine learning models for classification tasks

## Project Structure
- `data/` : sample datasets
- `notebooks/` : exploratory analysis and experiments
- `src/` : core data processing and modeling code
- `results/` : output figures and results

## Methods
- Data preprocessing and normalization
- Feature extraction from time-series data
- Classification using machine learning models (e.g., Random Forest)

## Future Work
- Improve model performance with deep learning
- Apply to larger astronomical survey datasets (e.g., LSST, Gaia)
